# Recipe Generator

A simple web app to generate recipes based on what's in your fridge. Built with plain HTML and powered by [OpenRouter](https://openrouter.ai).

#
# Tech Stack

- **Framework**: FastAPI
- **AI**: OpenAI SDK + OpenRouter API (model: `google/gemini-2.5-flash-lite`)
- **Templates**: Jinja2
- **Markdown render**: `markdown` library
- **Package manager**: `uv`
